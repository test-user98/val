# Where to add your memes & GIFs

Drop your files here. The site looks for these paths (create subfolders if needed):

- **Home:** `home/hero.gif`, `home/fun.gif`
- **Trivia:** `trivia/header.gif`, `trivia/q1.gif`, `trivia/q2.gif`, `trivia/q3.gif`
- **Valentine:** `valentine/ask.gif`, `valentine/shame.gif`, `valentine/yes.gif`
- **Birthday:** `birthday/cake.gif`
- **One more:** `onemore/header.gif`
- **Truth/Dare:** `truthdare/header.gif`, `truthdare/truth.gif`, `truthdare/dare.gif`

To use a different path, edit `index.html` (change the `src` in the meme-slot) or `config.js` (for trivia question memes).
